My Personal Project using Spotify and Ticketmaster APIs

In the summer of 2023, I and two peers created a web application that allows a user to sign in with their Spotify account and will then recommend concerts on Ticketmaster based on their musical preferences

To build this, I worked with two peers to develop a Flask web app that recommends concert tickets based on a user's Spotify listening history, top artists, and related artists using the Spotify API and Ticketmaster API

We implemented Python Pickle serialization/deserialization techniques to optimize data storage and retrieval, reducing the frequency of Ticketmaster API calls as the user base scaled, resulting in greatly improved program efficiency.

And we utilized session management to preserve login states across web pages while maintaining security through OAuth 2.0 implementation for protected user authentication

Disclaimer: the app won't work for the general public because we're still waiting on approval from Spotify for the project

Here my contributions included establishing connectivity from Business Application Studio to SAP Hana Cloud using Node.js, creating a Cloud Foundry App for users to interact with the Hana Cloud database, and implementing a feature that enables users to upload CSV files from their local drives to the cloud database.

I have completed various projects during school and hackathons, but my favorite is a personal passion project called “Tunes to Tickets”, a Python-based web application built on the Flask framework. It recommends concerts from Ticketmaster based on a user’s Spotify listening history and musical preferences, making use of both Spotify and Ticketmaster APIs. The project’s efficiency is enhanced through Python Pickle serialization for optimized data storage and retrieval which allows it to run faster as the user base scales. Moreover, it ensures user security via OAuth 2.0 while maintaining session management for seamless login experiences.
